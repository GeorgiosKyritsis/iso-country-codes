iso-country-codes
=================

### About

A Python dictionary of 246 [official ISO 2-digit country codes](http://www.iso.org/iso/english_country_names_and_code_elements) and the corresponding country names.

### Usage

		> from iso_country_codes import COUNTRY
		> print COUNTRY["GR"]
		GREECE
